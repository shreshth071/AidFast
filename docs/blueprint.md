# **App Name**: AidFast

## Core Features:

- Instruction Display: Display step-by-step first aid instructions in a clear, concise manner.
- Category Organization: Categorize first aid guides by type of injury or emergency for quick access.
- Interactive Checklists: Enable users to check off steps as they complete them in each guide.
- Real-time Search: Provide a search bar to quickly find relevant first aid guides based on keywords.
- Responsive Design: Ensure the interface adapts smoothly to desktop and mobile devices.
- AI-Powered Guide Suggestions: Tool: AI suggests related guides based on the user's current selection or search query, incorporating additional pieces of information if available.
- Voice Assistance: Voice assistance providing hands-free instructions for each step, integrating an audio walkthrough to enhance usability in critical situations.

## Style Guidelines:

- Primary color: Sky Blue (#87CEEB) to inspire a sense of calm and trust during emergencies.
- Background color: Very light blue (#E0FFFF) provides a clean and non-intrusive backdrop.
- Accent color: Soft Orange (#FFB347) for interactive elements and to highlight crucial steps.
- Body and headline font: 'PT Sans', a humanist sans-serif font to ensure readability and modernity.
- Use clear, universal icons for each category of first aid to enhance quick identification and navigation.
- Implement collapsible Bootstrap cards for well-organized content and responsive adaptation.
- Use subtle animations and transitions to guide the user through the steps, avoiding any unnecessary distractions.
