# 🚑 AidFast – Your Quick First Aid & Emergency Guide

**AidFast** is a responsive, user-friendly web application that helps users quickly access **emergency services, first aid instructions, and nearby medical help** in India. It is designed for urgent, life-saving scenarios where time and clarity matter most.

---

## 🌟 Features

### 🏠 Home – Emergency Contacts
- Direct emergency numbers for:
  - 🚑 Ambulance / Medical (Call 102)
  - 🚒 Fire Department (Call 101)
  - 🚓 Police (Call 100)

### 📚 Guides – First Aid & Disaster Response
- Interactive guides for common injuries:
  - Cuts and Scrapes
  - Minor Burns
  - Sprains
  - Nosebleeds
- Each guide includes an optional YouTube video for educational purposes.

> ⚠️ **Disclaimer:** All YouTube videos integrated into AidFast are **strictly for educational use**. AidFast does **not promote** or **monetize** this content in any way.

### 📍 Locator – Find Nearby Emergency Help
- Easily locate:
  - Hospitals
  - Police Stations
  - Fire Stations
  - Poison Control
  - Relief Camps
- Includes address and **"View on Map"** functionality.
